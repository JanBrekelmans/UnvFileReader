# UnvFileReader
UnvFileReader is a C++20 library for reading the Ideas-UNV mesh format.

# Getting Started
## Using CMake
This library can be easily added using CMake. An example shown below uses [CPM](https://github.com/cpm-cmake/CPM.cmake).
